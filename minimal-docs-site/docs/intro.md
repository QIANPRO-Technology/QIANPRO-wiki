# 千鉑Wiki

千鉑科技內部知識庫 — 教材、手冊、工作流程、開發筆記。

## 實戰課程

### 🧠 [Agent 實戰課程](./ai-agents/overview)

以 **LangChain / LangGraph** 為主軸，10 章教材從 Agent 概念到 DGX Spark 本地 vLLM 部署。適合有 Python 基礎、想學會做 AI Agent 的工程師。

**涵蓋**：Agent 概念、Tool Use、LangGraph、Memory、Human-in-the-Loop、Multi-Agent、RAG、生產部署。

---

### 🔌 [EdgeAI 實戰課程](./ai-sensor-mpu6050/overview)

以 **Raspberry Pi Pico 2 W + MPU6050 三軸 IMU** 為載具的 AIoT 完整鏈路：硬體接線、資料擷取、Edge Impulse 訓練、裝置端推論、Wi-Fi/MQTT 上傳。

**涵蓋**：硬體接線、MicroPython 環境、Edge Impulse 模型訓練、裝置端即時推論、MQTT 雲端上傳。

---

### 👁️ [Vision 實戰課程](./computer-vision-course/overview)

6 日課程從 **OpenCV** 一路到 **YOLO11 微調** 與 **CustomTkinter GUI** 整合，最後封裝成桌面端智慧視覺檢測工具。

**涵蓋**：OpenCV 基礎、Mediapipe 姿勢辨識、YOLO11 訓練、高解析影像處理、GUI 整合、PyInstaller 打包。
